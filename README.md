## CLPlayerDemo
An app to listen to a Casio drum loop on repeat for as long as you'd like.
##
1. Clone repo.
2. Dependencies are managed with CocoaPods. The Pods folder is checked into the repo so no `pod install` is necessary.
3. Run and enjoy!
